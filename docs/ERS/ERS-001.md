# MeetingHub-4
## Engineering Requirements Specification (ERS-001)

Version: 1.0
Status: Final — requisitos atendidos pelo hardware v1.0

---

# 1. Objetivo

Desenvolver um equipamento analógico capaz de conectar quatro notebooks simultaneamente utilizando headset TRRS padrão CTIA, permitindo:

- monitoramento de áudio das quatro fontes;
- comunicação por voz em apenas um notebook por vez;
- seleção rápida da origem do microfone;
- operação sem software ou drivers.

---

# 2. Requisitos Funcionais

## RF-001 - Entradas

O equipamento deverá possuir quatro entradas TRRS CTIA 3,5 mm para conexão direta com notebooks.

## RF-002 - Headset

O equipamento deverá utilizar um headset único TRRS CTIA.

## RF-003 - Monitoramento

O usuário deverá ouvir simultaneamente o áudio dos quatro notebooks.

## RF-004 - Controle de volume

Cada notebook deverá possuir controle individual de volume.

Deverá existir controle geral de volume.

## RF-005 - Microfone

O usuário deverá selecionar qual notebook receberá o sinal do microfone.

## RF-006 - Mute

O equipamento deverá possuir função MUTE física.

---

# 3. Compatibilidade

O notebook deverá interpretar o equipamento como um headset TRRS CTIA conectado diretamente.

Não deverá ser necessário:

- instalação de drivers;
- software adicional;
- configuração especial.

---

# 4. Requisitos de Durabilidade

O equipamento deverá ser projetado para uso intenso.

Meta:

- > 1 milhão de operações do seletor de microfone;
- conectores metálicos de painel;
- componentes industriais.

---

# 5. Restrições

Não utilizar:

- Bluetooth;
- USB áudio;
- firmware;
- processamento digital;
- aplicativos.

---

# 6. Alimentação

Entrada:

USB-C 5V.

Uso:

- mixer ativo;
- amplificador de headphone.

---

# 7. Critério de aprovação

O protótipo será aprovado quando:

- quatro notebooks funcionarem simultaneamente;
- o headset for reconhecido normalmente;
- o microfone funcionar no notebook selecionado;
- não houver necessidade de intervenção no Windows.
