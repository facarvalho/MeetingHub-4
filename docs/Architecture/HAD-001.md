# 1. Visão Geral

O MeetingHub-4 é um equipamento analógico destinado ao gerenciamento de áudio de quatro notebooks utilizando conexões headset TRRS CTIA.

O equipamento deverá:

- receber áudio de quatro notebooks;
- misturar os sinais em estéreo;
- disponibilizar monitoramento através de um único headset;
- direcionar o microfone do headset para apenas um notebook selecionado.

# 3. Filosofia de Projeto

O equipamento deverá ser transparente para os notebooks.

O computador deve interpretar o equipamento como um headset TRRS CTIA conectado diretamente.

Não serão utilizados:

- drivers;
- comunicação USB de áudio;
- processamento digital.


# 4. Subsistemas


## 4.1 Entrada TRRS

Cada notebook será conectado através de conector P2 TRRS CTIA metálico de painel.

Sinais:

- Tip: Audio Left
- Ring 1: Audio Right
- Ring 2: Ground
- Sleeve: Microfone


## 4.2 Sistema de Áudio

Arquitetura:

- mixer estéreo ativo;
- controle individual de volume;
- controle master;
- amplificador dedicado para headset.


## 4.3 Sistema de Microfone

Requisitos:

- preservação da polarização do microfone fornecida pelo notebook;
- chaveamento transparente;
- ausência de eletrônica ativa na linha do microfone.

Solução inicial:

- chaveamento através de relés de sinal.


## 4.4 Alimentação

Entrada:

USB-C 5V.

Alimentará:

- mixer ativo;
- amplificador de headphone;
- circuito de controle.


# 5. Considerações de Ruído

O projeto deverá considerar:

- aterramento em estrela;
- separação entre áudio e alimentação;
- filtros de alimentação;
- proteção contra interferência.
