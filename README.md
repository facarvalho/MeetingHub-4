# MeetingHub-4

## Overview

MeetingHub-4 is an analog audio management device designed to connect four computers using CTIA TRRS headset interfaces.

The system allows:

- simultaneous monitoring of four audio sources;
- selection of one microphone source;
- operation without drivers or software.
